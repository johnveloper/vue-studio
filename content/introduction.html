Introduction

Over the last 10 years our web pages have become more dynamic and powerful thanks to JavaScript.

We've moved a lot of code that was normally on the server side into our browsers, leaving us with thousands of lines of JavaScript code, connecting to various HTML and CSS files, with no formal organization.

This is why more and more developers are using JavaScript frameworks like Angular, React or Vue.

What is Vue?

Vue.js is an approachable, versatile, performant, web framework that helps you create a more maintainable and testable codebase.

Vue is a progressive framework, which means that if you have an existing server-side app, you can plug Vue into just one part of your app that need a richer, more interactive experience.

If you want to build more business logic into your front end from the get go, Vue has the core libraries and the ecosystem (Vue, VueX, Vue-Router) you'll need to scale.

Vue allows you to split your webpage into reusable components, each having its own HTML, CSS and JavaScript code.
